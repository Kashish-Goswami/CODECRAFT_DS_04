# CODECRAFT Data Science Internship - Task 04

## 📌 Objective

Analyze and visualize sentiment patterns in social media data to understand public opinion towards specific topics or brands.

---

## 📂 Dataset

- Source: Kaggle
- Dataset: Twitter Entity Sentiment Analysis
- Classes:
  - Positive
  - Negative
  - Neutral
  - Irrelevant

---

## 🛠 Steps Performed

1. Data Loading
2. Data Cleaning
3. Stopword Removal
4. TF-IDF Vectorization
5. Train-Test Split
6. Logistic Regression Model Training
7. Model Evaluation
8. Sentiment Visualization (WordCloud & Distribution)

---

## 📊 Model Performance

- Accuracy: **68%**
- Evaluation Metrics:
  - Precision
  - Recall
  - F1-score
  - Confusion Matrix

---

## 📈 Key Insights

- Negative sentiment was predicted most accurately.
- Logistic Regression performs well for multi-class text classification.
- WordCloud visualization reveals dominant keywords in positive sentiment.

---

## 🔧 Technologies Used

- Python
- Pandas
- NLTK
- Scikit-learn
- TF-IDF
- Logistic Regression
- Matplotlib
- WordCloud
